## general

| shortcut | action |
|---|---|
| <kbd>Cmd+,</kbd> | Settings |
| <kbd>Cmd+P</kbd> | Command palette |
| <kbd>Cmd+B</kbd> | Toggle Navigator |
| <kbd>Cmd+N</kbd> | New chat |
| <kbd>Cmd+T</kbd> | New terminal tab |
| <kbd>Cmd+W</kbd> | Close tab |
| <kbd>Cmd+O</kbd> | Open file |

## navigation

| shortcut | action |
|---|---|
| <kbd>Cmd+[</kbd> | Back in focused pane history |
| <kbd>Cmd+]</kbd> | Forward in focused pane history |
| <kbd>Ctrl+Tab</kbd> | Next chat session |
| <kbd>Ctrl+Shift+Tab</kbd> | Previous chat session |
| <kbd>Option+Cmd+→</kbd> | Next activity (editor focused: next document tab) |
| <kbd>Option+Cmd+←</kbd> | Previous activity (editor focused: previous document tab) |

## editor

|---|---|

| <kbd>Shift+Cmd+S</kbd> | Save as |

| <kbd>Cmd+I</kbd> | Italic (Markdown) |
| <kbd>Shift+Cmd+X</kbd> | Strikethrough (Markdown) |

## review & suggestions

| shortcut | action |
|---|---|
| <kbd>Cmd+Enter</kbd> | Accept change (diff review) |
| <kbd>Option+↑</kbd> | Previous chunk (diff review) |
| <kbd>Option+↓</kbd> | Next chunk (diff review) |
| <kbd>Tab</kbd> | Accept ghost suggestion |
| <kbd>Option+→</kbd> | Accept next word (ghost suggestion) |
| <kbd>Esc</kbd> | Dismiss review or suggestion |
