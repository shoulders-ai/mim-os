## Fast Orientation For A New Coding Agent

1. Read this file.
2. Run:

   ```bash
   npm run build
   npm run test
   ```

   When changing the app runtime contract, also run:

   ```bash
   npm run test:packages:compat
   ```

3. To launch the app:

   ```bash
   npm run dev
   ```

4. If you need AI features, ensure `~/.mim/keys.env` has at least one provider key configured.

## Architecture Overview

```
Electron App
  Main Process (Node.js)
    Boot (index.ts) → Tool Registry (`tools/registry.ts`) → Trace Log → Search DB → Server
    ├─ agents/ (CLI agent catalog/detection, agent session lifecycle, pty status tracker)
    ├─ ai/ (model registry, runtime, agent context, system prompt)
    ├─ packages/ (loader, manifests, enablement, runtime, jobs, data, HTTP, secrets)
    ├─ comments/ (inline markdown comment parser/model)
    ├─ resources/ (shared resource collection model)
    ├─ search/ (SQLite FTS, message text rows, file content search, text matching)
    ├─ security/ (main-process permission gate and path classifier)
    ├─ server/ (Express + WebSocket app/AI server)
    ├─ workspace/ (boot, mim.yaml contract, file watcher)
    ├─ tools/ (registry, fs, workspace, bridge, packages, search, settings)
    ├─ integrations/ (Slack, Google, keychain-backed secrets, HTTP boundary)
    ├─ headless.ts / cli.ts (non-Electron CLI registry)
    ├─ sessions.ts (CRUD, JSON file persistence)
    ├─ pty.ts (terminal spawning via node-pty)
    ├─ trace/ (unified trace stream: spans, identity, audit, payload blobs)
    └─ skills.ts (filesystem skill loader)

  Preload Bridge (preload/index.ts)
    window.kernel.call() ↔ ipcRenderer.invoke()

  Renderer (Vue 3 + Pinia)
    App.vue + services/appShell/ (Navigator / Work / Artifact compatibility shell)
    ├─ ChatView (AI SDK Chat + DefaultChatTransport to local AI runtime)
    ├─ EditorPanel (unified document tabs: CodeMirror text, PDF viewer, table grid, file cards)
    ├─ TerminalPanel (xterm.js + pty IPC)
    ├─ Workbench store (pane history, layout invariants, command path)
    ├─ Runs store (chat/app run aggregation)
    └─ Workbench components (NavigatorPane, WorkPane, ArtifactPane, hosts)

  App Iframes
    runtime.call() → SDK WebSocket → Server → Tool Registry
```
